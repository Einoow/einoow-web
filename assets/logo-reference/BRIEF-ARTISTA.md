# Brief: Recreación Logo EINOOW en SVG

## Objetivo
Recrear fielmente el logo "EINOOW" como archivo vectorial (SVG/AI) escalable.

## Referencia
- `einoow-logo-original.jpg` (434x188px) — única referencia disponible

## Elementos clave a replicar

### Tipografía
- Estilo: Fantasy/medieval con serifas decorativas
- Letras: E-I-N-O-W (la O es una gema)
- Efecto: Metálico dorado/bronce con degradados
- Borde: Contorno marrón oscuro para profundidad 3D

### La Gema (elemento central)
- Forma: Hexagonal (como esmeralda tallada)
- Color: Verde brillante con reflejos
- Posición: Reemplaza la "O" central
- Importancia: Es el elemento más memorable del logo

### Base/Pedestal
- Debajo de las letras hay una base decorativa tipo badge
- Color: Marrón/dorado coordinado

### Colores aproximados
- Oro/bronce: #C9A227 → #8B6914 (degradado)
- Verde gema: #4ADE80 → #166534 (con brillos blancos)
- Contorno: #3D2817

## Entregables requeridos

1. **SVG** — versión principal vectorial
2. **AI/EPS** — archivo editable Adobe Illustrator
3. **PNG transparente** — múltiples tamaños:
   - 512x512 (app icon)
   - 256x256
   - 128x128
   - 64x64 (favicon)
   - 32x32
4. **Variante solo gema** — para usar como icono/favicon
5. **Versión monocromática** — blanco y negro/escala de grises

## Notas
- Mantener fidelidad al original
- La gema debe funcionar como elemento independiente
- Priorizar que escale bien desde favicon hasta banner grande

## Uso previsto
- Web (einoow.com)
- App móvil (icono)
- Redes sociales
- Streams/videos
- Merch potencial
